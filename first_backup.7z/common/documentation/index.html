    JavaScript

    Important Javascript files:

    1. jQuery
    2. Thekamarel custom scripts
    3. Some plugins
    1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
    2. We have a thekamarel.js file where is all important scripts for example pages in theme. Every script have heading such as ->  
      /*============== CHOOSE SIDE WHERE SUBMENU OPEN ==============*/
    .click(function(evt) {
        var isRight = evt.pageX > (evt.view.outerWidth / 1.2);
        if (isRight === true) {
            $('.dropdown-submenu').addClass('submenu-left')
        }
        else {
            $('.dropdown-submenu').removeClass('submenu-left')
        }
    });
